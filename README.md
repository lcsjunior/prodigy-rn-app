# Prodigy IoT - Mobile APP

## Useful commands

Install packages:

```bash
yarn install
npm run prepare
```

Start development mode:

```bash
yarn android
# or
expo start
# or
F5
```

Connect to Reactotron:

```bash
adb reverse tcp:9090 tcp:9090
```

See: <https://github.com/infinitered/reactotron/blob/master/docs/quick-start-react-native.md>
